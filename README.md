# Diário de Humor

Aplicação web para registrar como você está se sentindo no dia.

## Tecnologias usadas

* Node.js
* Express
* HTML
* CSS
* JavaScript

## Funcionalidades

* Registrar humor
* Adicionar comentário
* Ver histórico
* Deletar registros

## Aplicação online

https://diario-humor.onrender.com

## Rodar localmente

1. Instalar dependências

npm install

2. Iniciar servidor

node backend/server.js
